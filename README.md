## A compiler written fully in NASM. 
The code includes:

* Lexical Analyzer
* Syntax Analyzer
* Semantic Analyzer
* Code Generator

The project was originally made for the sake of learning, working on it lasted an entire year. Started in 2019, finished in 2020 when I had to do my final year of highschool.

<b><code>full.asm</code></b> contains the 3 analyzers only. No code generator.

<b><code>cifullnew.asm</code></b> contains the code generator.

## Why was it made?

Aside from learning, I really liked the idea of low-level programming being easier. The idea was to make a compiler that gives ### **the control of assembly while being easy to use like python **. I also like a good challenge.
